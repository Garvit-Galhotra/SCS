# Rendering

1. SSR -> Server side rendering
   `HTML form data from backend to frontend`
   `Bad SEO then CSR`
   `Loads the HTML data from the backend`
   `Fast from CSR`

2. CSR -> Client side rendering
   `JSON data from backend to Frontend`
   `improves SEO`
   `Loads the JSON data from backend`
   `Slow from SSR`

# What is a Server ?

1. imagine a computer system (PC),
2. remove the monitor, keyboard, speaker, mouse
3. and the left is Computer Tower(Cabinet with the other components in it) with internet

- Computer Tower with internet is called as Server

# Axios (Tool)

1. Download it using `npm i axios`
2. import axios from 'axios'
3. axios.get('url') -> to get the data from the api
4. use async and await for fetching the data
   <Example-> 'https://picsum.photos/v2/list'>
